 Features

1. Add new tasks instantly

2. Mark tasks as completed

3. Delete tasks you don’t need

4. Responsive design (works on mobile, tablet, and desktop)

5 Simple and clean UI

6. Lightweight — built with pure JavaScript (no frameworks)



---

 Technologies Used

HTML5 – Structure of the app

CSS3 – Styling and layout

JavaScript (ES6) – Interactivity and task management logic


 Purpose

This To-Do List App is designed to help users stay organized by tracking:

1. Student study tasks

2. Daily household activities

3.  Work-related reminders

4. Personal goals and routines

 How to Use

1. Type your task into the input box

2. Click the Add button

3. Mark tasks as Completed when done

4. Delete tasks using the trash icon / delete button
